# React-Developer-Tools-chrome-extensions

Adds React debugging tools to the Chrome Developer Tools.

由于在国内无法连接 google chrome extensions,所以只好用本地的扩展程序代替了

在 chrome 中加载已解压的扩展程序,选择此文件夹即可
